Forms

Forms are used for user input fields. They include text inputs, number inputs, text areas, dropdowns, and checkboxes.

Text Input

Text inputs are simple input field that let user input one line of text. Basic HTML will do!

You can add errors by adding the class “has-error” to reduce the bottom margin of the text field.
